

# Gamified Quiz Center 

    A modern, interactive quiz platform with gamification elements designed to boost student engagement and learning outcomes.
    
# Quick Start
    # Clone the repository
    git clone https://github.com/todoor17/UID_Quiz_Assignment.git
    cd UID_Quiz_Assignment
    
    # Install dependencies
    npm install
    
    # Start development server
    npm run dev

    Open http://localhost:3000 in your browser.
# Features

   ## For Students

   - Take timed quizzes with instant feedback
   - View detailed results and performance analytics
   - Retry incorrect questions for targeted practice
   - Generate random practice quizzes
   - Compete on leaderboards
   - Export results to Excel
  ## For Teachers
 
  - Create and manage quizzes with intuitive editor
  - Enroll students and track their progress
  - Access detailed analytics and insights
  - Control quiz visibility and access
  - Use templates for quick quiz creation

  ## For Admins

  - Manage teacher accounts and classes
  - Configure platform settings
  - Monitor platform-wide usage

  ## Gamification Elements

  - XP Points - Earn experience for quiz completion
  - Leaderboards - Compete with classmates
  - Progress Tracking - Visualize your improvement
  - Levels - Unlock achievements through participation
 

  ## Tech Stack

  - Frontend: React + TypeScript
  - Styling: CSS3 with modern design patterns
  - Build Tool: Vite
  - UI/UX: Task-centered design principles

  ## Responsive Design
   ### Works seamlessly on:

   - Desktop computers
   - Mobile phones
   - Tablets
   - Any screen size

# Key Workflows
 ## Student  

   Login → View Classes → Select Quiz
   Take Quiz → Submit Answers
   View Results → Retry Incorrect Questions
   Track Progress → Check Leaderboard

## Teacher 

   Login → Access Dashboard
   Create/Edit Quiz → Enroll Students
   Monitor Performance → View Analytics
